## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Environment Variables

Create a `.env` file in the root directory:

```
VITE_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key
```

## Tech Stack

- React 18
- Vite
- React Router
- Stripe.js
- Supabase (for backend)
